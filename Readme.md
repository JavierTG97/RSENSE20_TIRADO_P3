# Práctica 3: Comunicación, gestión y representación de datos de sensores con Python

### *Redes de sensores Electrónicos*

**Partes de la práctica** (cada una tiene un folder asociado):

1. **Serial**: Programa en Python un programa que acceda al puerto serie y muestre por pantalla los datos en tiempo real (igual que el terminal). Necesitarás instalar el objeto pyserial como administrador.
2. **FicheroTXT**: Modifica el programa para que almacene los datos en fichero .txt separando cada variable con “;” y con retorno de carro al final de cada muestra para abrirlo desde Excel.
3. **Gráficas**: Modifica el programa para que acumule los datos durante 5 segundos, calcule el promedio y desviación estándar los represente en tiempo real.
